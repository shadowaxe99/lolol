# Changelog

## Version 1.0.0 (2022-01-01)

### Added
- Initial release of Sara, an AI-powered executive assistant.
- Task Management feature using Google Tasks API and Microsoft To Do API.
- Schedule Management feature using Google Calendar API and Microsoft Outlook Calendar API.
- Email Management feature using Gmail API and Microsoft Graph API.
- Customization feature using machine learning algorithms.
- Performance Metrics to gauge Sara's performance.
- Dependencies on OpenAI's GPT-3.5 model and various APIs.

## Version 1.1.0 (2022-02-01)

### Added
- Graphical User Interface (GUI) for Sara's web application.
- Interactive Chat Interface for direct interaction with Sara.
- Dashboard for an overview of tasks, schedule, and emails.
- Task Management Interface for creating, updating, and deleting tasks.
- Schedule Management Interface with an interactive calendar view.
- Email Management Interface resembling a traditional email client.
- Customization and Settings page for user preferences.
- User Authentication with login, register, and password recovery options.
